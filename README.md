# 旅行業界 用語集1000

旅行業界で使用される専門用語を検索・閲覧できるWebアプリケーションです。

## 概要

このプロジェクトは、旅行業界の基礎知識、法規、航空、宿泊などのカテゴリーに分類された用語集を提供します。ユーザーはキーワードで用語を検索したり、ランダムに表示された用語から新しい知識を得ることができます。

## 機能

- **用語検索**: 用語名、読み仮名、意味、カテゴリーからキーワード検索が可能です。
- **ランダム表示**: ページ読み込み時にランダムに用語をピックアップして表示します（PC: 12件、SP: 6件）。
- **詳細表示**: 用語カードをクリックすると、モーダルウィンドウで詳細（読み、カテゴリー、意味）を確認できます。
- **レスポンシブデザイン**: スマートフォンとPCの両方で見やすいレイアウトを採用しています。

## ファイル構成

- `index.html`: メインページ
- `style.css`: アプリケーションのスタイルシート
- `script.js`: アプリケーションロジック（検索、表示、モーダル制御など）
- `data/data.js`: 用語データ格納ファイル

## データ構造

`data/data.js` 内の `allTerms` 配列に以下の形式でデータが格納されています。

```javascript
{
    id: 1,
    term: "募集型企画旅行",
    reading: "ぼしゅうがたきかくりょこう",
    meaning: "...",
    category: "01. 基礎・法規・約款"
}
```

## 使用方法

1. `index.html` をブラウザで開いてください。
2. 検索ボックスにキーワードを入力して「検索」ボタンを押すか、Enterキーを押すと検索結果が表示されます。
3. カードをクリックすると詳細が表示されます。
